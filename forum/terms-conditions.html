Användarvillkor

Dessa användarvillkor reglerar användningen av Enatist Community-plattformen. Genom att besöka eller använda plattformen godkänner användare att följa dessa villkor. Om en användare inte accepterar villkoren ska användningen av plattformen upphöra.

Plattformens syfte

Enatist-plattformen tillhandahåller en digital miljö där medlemmar kan delta i community-diskussioner, skapa inlägg, dela insikter och interagera med andra medlemmar. Plattformen inkluderar även ett bidragssystem där användare tjänar badges, bidragspoäng och rankningar baserat på sin aktivitet.

Plattformens mål är att främja meningsfullt deltagande, kunskapsdelning och respektfull interaktion mellan community-medlemmar.

Användarkonton

Användare kan behöva skapa ett konto för att delta i diskussioner, skapa inlägg eller tjäna bidragspoäng. Varje användare ansvarar för att hålla sitt kontouppgifter konfidentiella och för alla aktiviteter som sker under kontot.

Användare ska lämna korrekta uppgifter vid registrering och får inte utge sig för att vara någon annan person eller organisation.

Deltagande i communityn

Medlemmar uppmuntras att bidra på ett respektfullt och konstruktivt sätt. Inlägg, kommentarer och diskussioner får inte innehålla skadligt, stötande, vilseledande eller olagligt innehåll.

Plattformen förbehåller sig rätten att ta bort innehåll eller begränsa användares tillgång om aktiviteter bryter mot community-regler eller stör miljön för andra medlemmar.

Badge- och bidragssystem

Plattformen har ett system där medlemmar tjänar badges baserat på antalet inlägg. Dessa badges kan ge bidragspoäng som bidrar till användarens placering i community-rankningen.

Bidragspoäng och rankningar beräknas automatiskt baserat på aktivitet på plattformen. Medlemmen med högst poäng kan visas som den högst rankade bidragsgivaren i communityn.

Plattformen kan uppdatera eller justera badge-systemet, bidragspoängen eller rankningsmetoderna vid behov för att förbättra community-upplevelsen.

Innehållsägande

Användare behåller äganderätten till det innehåll de skapar på plattformen. Genom att publicera innehåll ger användare dock plattformen rätt att visa, distribuera och lagra detta innehåll inom community-miljön.

Användare får inte publicera innehåll som kränker immateriella rättigheter eller andras rättigheter.

Tillgänglighet

Plattformen strävar efter att vara tillgänglig och fungerande, men kan inte garantera oavbruten tillgänglighet hela tiden. Underhåll, tekniska problem eller systemuppdateringar kan tillfälligt påverka åtkomsten till plattformen.

Uppsägning av åtkomst

Plattformen förbehåller sig rätten att suspendera eller avsluta användarkonton om användaren bryter mot dessa villkor eller ägnar sig åt aktiviteter som skadar communityn eller plattformen.

Uppdateringar av villkoren

Dessa användarvillkor kan uppdateras periodvis för att återspegla förändringar i plattformens funktioner, policyer eller lagkrav. Användare uppmuntras att regelbundet granska denna sida.
